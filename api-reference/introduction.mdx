## Base URL

All endpoints are served from:

```
https://mandrillapp.com/api/2.0
```

## How this reference works

Every page in this section is generated directly from our [OpenAPI specification](/openapi.json).
When the spec changes, these pages regenerate automatically — the reference can never
drift from the live API.

<Info>
  Use the **API playground** on any endpoint page to send real requests and see live
  responses without leaving the docs.
</Info>
